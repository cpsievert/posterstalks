  • If the gov't opens up data we should try to make useful apps with that data

TaxaViewer Description


TaxaViewer takes in species names and outputs:

  • Taxonomic information

  • Taxonomic classfication

  • Downstream taxa

  • Phylogeny (plants only) with invasive status mapped on

  • Map of occurrences

  • Related papers in PLOS (Public Library of Science) journals

TaxaViewer Features


Some important features:

  • Simple interface, especially since submission

  • New interactive map interface

  • Some content can be downloaded

  • Perhaps most important Black code buttons provide R code to reproduce locally what is done in the app

Technology

  • Interface is Shiny, created by RStudio, a wrapper around R code to add html, javascript, etc.
    • Note that Shiny apps can be run locally and on a server on the web
    • When run locally or on web, rendered in browser
  • App hosted on Glimmer, servers hosted by RStudio at no cost
  • Backend - all web API calls, data munging, and plotting - done in R
  • Styling using Bootstrap
  • Icons by FontAwesome
  • Interactive map using rCharts by Ramnath Vaidyanathan and leaflet.js
